# AI Drama Production — AI 剧集制作技能

一个适用于 Claude Code 的 AI 剧集与漫剧制作技能。双模式，同一框架：

- **🎬 电影模式** — 基于 Seedance/即梦 的连续视频生成工作流，适用于 AI 短片、预告片、电影级片段
- **🎞️ 漫剧模式** — 旁白驱动的静态面板装配工作流，适用于即梦出图 + 剪映合成，专注于叙事型故事和写实/科幻质感

---

## 功能特性

- **双生产模式** — 接入时选择电影或漫剧模式，后续工作流自适应
- **创意层共享** — 世界观构建、角色设定、场景圣经、风格圣经，两种模式通用
- **导演讲戏** — 将抽象情绪转化为可见动作链（电影模式）
- **旁白映射** — 自动字数-时长估算，旁白逐句与画面面板匹配（漫剧模式）
- **批量图生 Prompt** — 风格前缀自动注入、角色参考图自动关联
- **运镜分配** — 漫剧模式 4 种运镜模板、电影模式完整镜头语法
- **剪映装配指南** — 每面板时长、关键帧动画、转场、多轨道时间线布局
- **审查清单** — 7+ 评分关卡，PASS/WARN/FAIL 判定 + 修复策略
- **纯 Markdown 输出** — 不需要后端、数据库、登录

---

## 双模式对比

| | 电影模式 | 漫剧模式 |
|---|---|---|
| **适合场景** | AI 短片、预告片、电影级片段 | 叙事型故事、写实/科幻质感漫剧 |
| **生成方式** | 连续视频（每段 4-15 秒） | 静态图片 + 关键帧动画装配 |
| **工具链** | Seedance / 即梦 图生视频 | 即梦 出图 + 剪映 装配 |
| **核心产出** | 视频提示词包、分镜表 | 面板映射表、批量图生 Prompt、装配指南 |
| **关键能力** | 导演讲戏（讲戏本）、镜头连续性 | 旁白节奏、面板一致性、流程控制 |
| **生产速度** | 较慢：逐镜头生成视频 | 快速：批量出图 + 一次性装配 |

## 安装

### 作为 Claude Code 技能使用

```bash
# 将仓库克隆到 Claude Code 的技能目录
git clone https://github.com/Amouren7/ai-drama-production.git \
  ~/.claude/skills/ai-drama-production
```

或者直接将 `ai-drama-production` 文件夹复制到 `~/.claude/skills/` 目录下。

### 独立使用

本技能的 Markdown 输出可以完全独立使用。生成后无需 Claude Code，直接按照提示词、装配指南和工作流说明操作即可。

## 快速开始

1. 在 Claude Code 中调用技能，描述你的项目：
   ```
   /ai-drama-production 我想做一个赛博朋克题材的AI短片
   ```
2. 技能会询问你要用什么模式以及几个项目基本信息。
3. 从概念到最终导出，按照生成的制作指南操作。

### 你需要准备

- 故事概念（两三句话也可以）
- 如有参考图则更佳（角色设计、场景参考）
- 目标平台和大致时长

## 工作流概览

### 电影模式

```
接入 → 四基石（世界观/角色/故事/风格）
     → 导演讲戏（讲戏本）
     → 分镜 + Seedance 提示词生成
     → 图片粗剪 → 迭代 → 声音 → 最终
```

完整工作流：`references/workflow.md`

### 漫剧模式

```
接入 → 四基石（与电影模式共享）
     → 旁白脚本 + 时间线拆解
     → 面板映射
     → 批量图生 Prompt 生成
     → 运镜分配
     → 剪映装配指南
     → 声音计划 → 审查 → 导出
```

完整工作流：`references/workflow-motion-comic.md`

## 工具链要求

### 电影模式

- **图片/视频生成：** Seedance 2.0 或 即梦
- **剪辑：** 剪映、Premiere Pro 或其他非线性编辑软件

### 漫剧模式

- **图片生成：** 即梦（或兼容的图生图工具）
- **装配：** 剪映（关键帧动画、多轨音频）
- **配音：** 剪映内置配音、第三方 TTS、或真人录音

## 核心设计原则

- **故事优先。** 在前提、冲突和情感上投入更多精力，而不是工具细节。
- **在设计时就锁死一致性。** 在写提示词之前先固化角色和场景参考图。
- **先做粗剪再出最终版。** 静态图片粗剪能以极低成本发现节奏和构图问题。
- **声音就是叙事。** 配音、环境音、音效、音乐各司其职。
- **产出物可直接进入生产。** 每份文档从素材到装配都是完整的操作指南。
